# my_pvz

基于 C++ 与 FTXUI 实现的终端植物大战僵尸游戏。

## 项目介绍

本项目是 AI 驱动开发实践课程的大作业。

使用：

* C++
* FTXUI
* CMake
* GitHub

开发了一个终端版本的植物大战僵尸游戏。

项目实现了：

* 菜单系统
* 地图绘制
* 植物种植
* 僵尸移动
* 子弹攻击
* 阳光系统
* 游戏结束判定

## 编译方法

### Linux / WSL

mkdir build
cd build
cmake ..
make
./world

## 操作方式

| 按键    | 功能     |
| ----- | ------ |
| WASD  | 移动光标   |
| 1     | 选择豌豆射手 |
| 2     | 选择向日葵  |
| 5     | 选择坚果墙  |
| Space | 种植植物   |
| X     | 删除植物   |
| Enter | 查看植物属性 |
